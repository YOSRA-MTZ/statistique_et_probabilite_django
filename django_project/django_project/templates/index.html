à propos de l'application
Visualisation
Les utilisateurs auront la capacité de créer des graphiques et des diagrammes à partir des colonnes de données choisies. Ils pourront sélectionner le type de graphique (par exemple, histogramme, diagramme à barres, nuage de points) et personnaliser les paramètres de visualisation.
Parcours des fichier
Une fois les données chargées, les utilisateurs pourront parcourir les données, filtrer des colonnes spécifiques, trier les données, et effectuer des statistiques de base
